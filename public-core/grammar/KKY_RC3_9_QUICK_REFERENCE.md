# 깐따삐야어 RC3.9 빠른 참조

## 기본형

| 기능 | 형식 |
|---|---|
| 자동사 | `S-∅ V=루` |
| 타동사 | `A-스 P-∅ V=루` |
| 전달 | `A-스 THEME-∅ RECIPIENT=겨 코레=루` |
| 소유 | `POSSESSOR HEAD-CASE` |
| 부정 | `ROOT-CNEG NEG-TENSE=FINAL` |
| 부정 명령 | `ROOT-CNEG NEG=IMP` |

## 종결

| 종결 | 의미 |
|---|---|
| `=루` | 평서 |
| `=호` | 예/아니오 의문 |
| `=셴` | WH 의문 |
| `=얀` | 명령 |

## 관계절 head

| Head | 표지 |
|---|---|
| theme/P | bare ABS 또는 matrix case |
| FRC/interface | `HEAD-CASE=겨` |
| ordinary recipient | `HEAD-CASE=겨` |

핵심: `=겨`는 ERG/ABS를 대체하지 않는다. `에안-스=겨`는 가능하지만 `*에안=겨-스`는 불가하다.


## 011C 보강

- WH A는 `마사름-스 ... =셴`처럼 `-스`를 받는다.
- 타동 긍정 명령에서 overt addressee A도 `-스`를 받는다: `에안-스 열-실-∅ 열=얀`.
- `=헌`/`=두` edge clitic은 RC3.9 공개핵에서 `=루` 뒤에서만 실증한다.
